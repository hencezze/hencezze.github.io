LIGHTSAIL

Simple type of EC2 Instance on which you can bring up little service, for example WordPress, or can bring up your service via docker image,  but you will not able to make advance settings, this service for simple deploying containers and instances

Also you can define your containers to public, for this you should attach public endpoint

 

AWS Lambda

With this service you can add your own code without servercreating, you will pay only for time, when lambda works.

You can attach her to a lot of other AWS services via triggers, which will run lambda code, for instance: When you add picture to S3Bucket lambda will start and copy this picture into second S3

Or you can create simple landing site page, wich will give 200 response with index.html, if you want - you can write in more responses, for example on Python

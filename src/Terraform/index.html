Modules

If you want attach variables from main.tf to yours local or other module you should to define this variables in main.tf and define this variables in variables.tf file in this module without hardcoding value, but you can define only variable type

In main.tf                                                                          In variables.tf ( in module )

module "network" {                                                           variable "region" {

     source = "./modules/networks/"                                        type = string

     region = var.region                                                        }

     range = var.range

}

Also you can share variable from module, you should create output.tf on which you define variables, which can be attached into main.tf

In main.tf                                                                          In output.tf ( in module )

module "network" {                                                           output "ntwrk" {

     source = "./modules/VM/"                                                 value = path.to.value

     region = module.network.ntwrk                                         sensitive = true

     range = var.range                                                         }

}
